# OficinaMJS
# 🚗 Sistema de Gestão de Oficina (Em Desenvolvimento)

## 📌 Sobre o Projeto
Este projeto consiste no desenvolvimento de um sistema web para gestão de uma oficina automóvel.  
O objetivo é centralizar e otimizar processos como cadastro de clientes, controle de serviços, gestão de veículos e acompanhamento de ordens de serviço.

⚠️ **Status:** Em desenvolvimento ativo



## 🎯 Objetivos
- Digitalizar o fluxo de trabalho da oficina  
- Melhorar o controle de serviços e atendimentos  
- Facilitar a gestão de clientes e veículos  
- Criar uma base escalável para futuras funcionalidades
